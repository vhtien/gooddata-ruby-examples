=== Create an empty project
by Tomas Svarovsky

==== Problem
You need to create a project programmatically.

==== Solution

[source,ruby]
----
# encoding: UTF-8

require 'gooddata'

GoodData.with_connection('user', 'password') do |c|
  GoodData::Project.create(title: 'This is my new project', auth_token: 'TOKEN')
end
----
